<app-store>, <app-form>, <app-todos>

This example shows how to use parent factory to create simple application with shared state between custom elements.

Static in <app-root>

<app-store todos='["One", "Two"]'>
  <app-todos></app-todos>
  <app-form></app-form>
</app-store>

Use form to add todo to the list. Click on todo on list to remove it. Change default todos in <app-root> render method.
